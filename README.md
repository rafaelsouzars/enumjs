# **Enumjs**

Biblioteca javascript com uma classe __enumeradora__.


Exemplo: 
```javascript
let semana = new Enum(["SEG","TER","QUA","QUI","SEX","SAB","DOM"]);

console.log(`Domingo = ${semana.DOM} Terca = ${semana.TER}`);
console.log(semana.showAll());
```
